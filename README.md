# 🚀 SmartSort Organizer

SmartSort is a powerful Windows desktop application that automatically organizes your files into structured folders based on file type.

---

## ✨ Features

- 📂 Automatic file organization (all file types supported)
- 🖼 Image subfolders (JPG, PNG, GIF, etc.)
- 📁 Custom folder selection
- ⚡ Real-time background sorting
- 🖥 System tray support (runs in background)
- 🔄 Auto update system (download + restart)
- 🚀 Auto start with Windows
- 🎬 Smooth splash screen (Avro-style)
- 🧠 Smart detection (avoids moving incomplete downloads)

## 🛠 Installation

1. Run `SmartSortInstaller.exe`
2. Click **Next → Next → Install**
3. Launch SmartSort from Start Menu or Desktop

---

## ▶️ How to Use

### 1. Start Organizer
- Click **Start**
- App will begin organizing files automatically

---

### 2. Select Folder
- Click **Select Folder**
- Choose any folder you want to organize

---

### 3. Run in Background
- Click ❌ (close button)
- App will continue running in system tray

---

### 4. Enable Startup
- Click **Add to Startup**
- App will run automatically when Windows starts

---

### 5. Update App
- Click **Check for Updates**
- If new version available → auto download + restart

---

## ⚠️ Notes

  → Click **More Info → Run Anyway**
- App must be closed before updating
- Do not manually move update files
